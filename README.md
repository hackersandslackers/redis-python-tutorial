## Getting Started

Installation is recommended with Poetry:

```shell
$ git clone https://github.com/hackersandslackers/redis-python-tutorial.git
$ cd redis-python-tutorial
$ poetry shell
$ poetry install
$ poetry run
```

Alternatively, try installing via `setup.py`:

```shell
$ git clone https://github.com/hackersandslackers/redis-python-tutorial.git
$ cd redis-python-tutorial
$ python3 setup.py install
$ python3 main.py
```
